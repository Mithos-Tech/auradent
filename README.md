# 🦷 Varenne — Odontología de Élite & Diseño Dental de Vanguardia

> **Varenne by Inspyrio** es una landing page de estética y odontología digital de élite desarrollada con una experiencia de usuario (UX) impecable inspirada en los diseños más sofisticados de **Framer**. Este proyecto sirve como una demo interactiva de producción de alto rendimiento, optimizada para la conversión y preparada para integraciones modernas.

## 🎨 Características de Diseño & Estructura (Aesthetics Audit)

El desarrollo visual de Varenne está guiado por la filosofía "Framer-Minimalist", eliminando ruidos visuales innecesarios u ornamentos superfluos (anti-AI slop) y enfocándose en la jerarquía, el espacio negativo y las micro-interacciones:

*   **Bento Grid de Alta Precisión**: Una cuadrícula asimétrica y responsiva optimizada a un ancho máximo premium de `1440px` (específicamente calibrado para las secciones de **Servicios**, **Ventajas**, **Casos de Éxito** y **Preguntas Frecuentes**), asegurando legibilidad óptima y equilibrio estético en pantallas ultra-anchas y dispositivos móviles.
*   **Deslizador de Comparación Interactivo (Antes & Después)**: Slider interactivo nativo integrado que permite al paciente deslizar para visualizar el impacto real de los tratamientos odontológicos (Ortodoncia invisible, carillas, blanqueamiento premium).
*   **Acordeón de Preguntas Frecuentes (FAQ) de Doble Columna**: Estructura asimétrica sobre contenedor de contraste azul royal, diseñado para responder dudas clínicas sin fatiga visual.
*   **Footer de Estilo Framer Sofisticado**: Un pie de página rediseñado que elimina banners de suscripción redundantes (dado que la aplicación ya cuenta con sección de contacto y un chatbot interactivo). Ofrece accesos directos de navegación limpia, detalles de tecnología clínica, cinta de horarios de servicio y los créditos oficiales de autoría en perfecta armonía cromática oscura.
*   **Tipografía de Autor**: Integración premium de *Onest* para un tono corporativo confiable y sofisticado, complementado con *JetBrains Mono* para métricas clínicas de precisión.

## 🔍 SEO Profesional de Clase Mundial

Se ha implementado una arquitectura de posicionamiento en buscadores (SEO) integral y profesional directamente en la cabecera del sitio:

1.  **Metaetiquetas Semánticas**: Meta descripción atractiva, palabras clave enfocadas a audiencias premium y etiquetas de indexación completas.
2.  **Protocolo Open Graph & Twitter Cards**: Optimizado para previsualizaciones impecables en aplicaciones de mensajería (WhatsApp, Telegram, Slack) y redes sociales, utilizando imágenes de alta resolución e incentivos de conversión.
3.  **Datos Estructurados JSON-LD (Schema.org)**: Integración de un esquema tipo `Dentist` / `LocalBusiness` que define el nombre oficial, geolocalización precisa de la clínica en Av. Paseo de la Reforma, horarios de apertura (`openingHoursSpecification`), rango de precios, contacto directo y créditos al fundador de manera comprensible para el rastreador de Google.
4.  **Favicon de Alto Rendimiento**: Favicon SVG inline optimizado para pantallas retina que muestra el isotipo monogramático elegante de Varenne.

## 🛠️ Stack Tecnológico

*   **React 18** + **Vite** para una velocidad de respuesta instantánea.
*   **TypeScript** completo para control de tipos robusto.
*   **Tailwind CSS** para un control milimétrico del diseño responsivo.
*   **Framer Motion (`motion/react`)** para transiciones fluidas de estado, despliegue de FAQs y micro-animaciones en tarjetas.
*   **Lucide React** para iconografía minimalista de alta definición.

## 🚀 Despliegue & Ejecución Local

Para ejecutar o compilar este desarrollo para entornos de producción:

1.  **Instalar dependencias**:
    ```bash
    npm install
    ```
2.  **Iniciar servidor de desarrollo**:
    ```bash
    npm run dev
    ```
3.  **Compilar para producción (Build)**:
    ```bash
    npm run build
    ```
4.  **Auditar Estilo & Linter**:
    ```bash
    npm run lint
    ```
